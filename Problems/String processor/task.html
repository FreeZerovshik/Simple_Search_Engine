String processor

Write a program that emulates a string processor. It reads two strings and an operator and outputs the result.

There are three possible operations:

  • equals returns true if these strings are equals, otherwise false;
  • plus appends the second line to the first line;
  • endsWith returns true if the first string ends with the second string, otherwise false.

The program is to output the result as a string. If the operation is unknown, the program is to print "Unknown operation" without quotes.

You can use the string function endsWith(suffix) to find if the string ends with the specified string suffix:

val text = "Good morning"
println(text.endsWith("morning")) // true
println(text.endsWith("day")) // false

Sample Input:
abc
equals
cde

Sample Output:
false


Sample Input:
abbc
plus
bb

Sample Output:
abbcbb


Sample Input:
abbc
endsWith
bc

Sample Output:
true
