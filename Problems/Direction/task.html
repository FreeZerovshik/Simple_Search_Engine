Direction

Write a program, that reads the number of direction (1 – up, 2 – down, 3 – left, 4 – right, 0 – stay) and outputs the text "move up" (or "move down", or "move left", or "move right", or "do not move" depending on the number entered) without quotes. If the input does not correspond to any of the listed directions, output: "error!"

Note: the output text is to exactly match the sample! It is true for letters case and location of spaces as well.


Sample Input:
1

Sample Output:
move up



Memory limit: 256 MB
Time limit: 5 seconds
